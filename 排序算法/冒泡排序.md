## 冒泡排序

#### 基本描述

重复和相邻的元素进行比较，然后替换

#### 算法原理

````
待排序数组 [8, 10, 7, 5, 4, 3]
第一轮 
[8, 10, 7, 5, 4, 3]
[8, 7, 10, 5, 4, 3]
[8, 7, 5, 10, 4, 3]
[8, 7, 5, 4, 10, 3]
[8, 7, 5, 4, 3, 10]
第二轮
[7, 8, 5, 4, 3, 10]
[7, 5, 8, 4, 3, 10]
[7, 5, 4, 8, 3, 10]
[7, 5, 4, 3, 8, 10]
[7, 5, 4, 3, 8, 10]
......
````

#### 代码实现

```` java
int arr[] = new int[] {8, 10, 7, 5, 4, 3};
for (int x = 0; x < arr.length - 1; x ++) {

  for (int y = 0; y < arr.length - 1; y ++) {
    if (arr[y] > arr[y + 1]) {
      int temp = arr[y];
      arr[y] = arr[y + 1];
      arr[y + 1] = temp;
    }
  }
}
// 输出结果
for (int x = 0; x < arr.length; x ++)
            System.out.println(arr[x]);
````
